# 布局系统

EduPPTX V2 使用 Bento Grid 卡片布局系统。LLM 生成全页 SVG (1280×720)，系统转换为 PowerPoint 原生形状。

## 画布坐标

SVG viewBox: `0 0 1280 720` (16:9)

转换为 PowerPoint EMU:
```
宽度: 12,192,000 EMU (1280 × 9525)
高度:  6,858,000 EMU (720 × 9525)
1 SVG px = 9525 EMU (96 DPI)
```

## 画布分区

```
┌─────────────────────────────────────┐ y=0
│  页面标题区 (y: 30-90)               │
│  主标题 x=50, y=50, font-size=32    │
│  副标题 x=50, y=90, font-size=16    │
├─────────────────────────────────────┤ y=110
│                                     │
│  卡片内容区 (y: 110-660)             │
│  左边距 x=50, 右边距 x=1230          │
│  可用宽度 = 1180                     │
│  可用高度 = 550                      │
│                                     │
│  卡片间距 = 20px                     │
│  卡片圆角 = 12-16px                  │
│  卡片内边距 ≥ 24px                   │
│                                     │
├─────────────────────────────────────┤ y=670
│  页脚区 (y: 680-710)                 │
│  页码 x=1220, y=700, font-size=12   │
└─────────────────────────────────────┘ y=720
```

## Bento Grid 布局模式

### center_hero
单张大卡片居中。适用于封面、标题页、定义页。
```
┌─────────────────────┐
│                     │
│    ┌───────────┐    │
│    │  BIG CARD │    │
│    └───────────┘    │
│                     │
└─────────────────────┘
```

### vertical_list
纵向等宽卡片堆叠。适用于目录、步骤序列。
```
┌─────────────────────┐
│ ┌─────────────────┐ │
│ │ Item 1          │ │
│ ├─────────────────┤ │
│ │ Item 2          │ │
│ ├─────────────────┤ │
│ │ Item 3          │ │
│ └─────────────────┘ │
└─────────────────────┘
```

### bento_2col_equal
两等分列。适用于对比、优缺点分析。
```
┌──────────┬──────────┐
│          │          │
│  Card A  │  Card B  │
│          │          │
└──────────┴──────────┘
```

### bento_2col_asymmetric
非对称两栏 (约 2:1)。左侧主内容，右侧补充。
```
┌──────────────┬─────┐
│              │     │
│   Main       │Side │
│              │     │
└──────────────┴─────┘
```

### bento_3col
三等分列。适用于三要素并列。
```
┌───────┬───────┬───────┐
│       │       │       │
│ Col 1 │ Col 2 │ Col 3 │
│       │       │       │
└───────┴───────┴───────┘
```

### hero_top_cards_bottom
上方大卡 + 下方 2-4 小卡。适用于概述+细节。
```
┌─────────────────────┐
│      Hero Card      │
├──────┬──────┬──────┤
│ Sm 1 │ Sm 2 │ Sm 3 │
└──────┴──────┴──────┘
```

### hero_with_microcards
一张外层大卡 + 卡内 2-6 个微模块。适用于识字卡、诗文分栏、重点讲解等需要“主卡统领，卡内再分区”的页面。
```
┌─────────────────────────────┐
│         BIG HERO CARD       │
│ ┌─────┐ ┌─────┐ ┌─────┐     │
│ │ M1  │ │ M2  │ │ M3  │     │
│ ├─────┤ ├─────┤ ├─────┤     │
│ │ M4  │ │ M5  │ │ M6  │     │
└─────────────────────────────┘
```

### mixed_grid
自由组合大小卡片。适用于复杂知识点。
```
┌──────────┬─────┐
│  Large   │ Sm1 │
│          ├─────┤
├─────┬────┤ Sm2 │
│ Sm3 │Sm4 │     │
└─────┴────┴─────┘
```

### relation layout_hint
中心节点 + 分支节点 + 连接线/箭头。适用于 `content` 页中的概念关系、因果关系、分类结构等关系图布局。
```
      ┌───────┐
      │ Center│
  ┌──→└───────┘←──┐
┌─────┐       ┌─────┐
│Node1│       │Node2│
└─────┘  ─→   └─────┘
     ┌─────┐
     │Node3│
     └─────┘
```

## SVG→DrawingML 转换

### 形状映射

| SVG | DrawingML | 备注 |
|-----|-----------|------|
| `<rect>` | `<a:prstGeom prst="rect">` | 无圆角 |
| `<rect rx="14">` | `<a:prstGeom prst="roundRect">` | avLst val 计算: rx/min(w,h)×50000 |
| `<circle>` | `<a:prstGeom prst="ellipse">` | cx,cy,r → x,y,w,h |
| `<text>` | `<p:sp txBox="1">` | 单行: wrap="none", 多行: wrap="square" |
| `<path d="...">` | `<a:custGeom>` | M→moveTo, L→lnTo, C→cubicBezTo, Z→close |
| `<image>` | `<p:pic>` | base64 提取到 ppt/media/ |

### 文本框处理

单行文本 (`<text>` 无 tspan):
- `wrap="none"` + `<a:spAutoFit/>`
- 宽度由内容自动撑开

多行文本 (`<text>` 含 tspan):
- `wrap="square"` (启用自动换行)
- 宽度从 tspan x 属性推断卡片边界
- 每个 tspan (dy>0) 转为独立段落 `<a:p>`
- 行距: 基于 dy 值设置 `<a:lnSpc>/<a:spcPts>`

### 字体映射

```
font-family="Noto Sans SC, 微软雅黑, Arial, sans-serif"
→ <a:latin typeface="Arial"/>
  <a:ea typeface="Noto Sans SC"/>
```

自动检测 EA 字体（PingFang SC, Microsoft YaHei, Noto Sans SC 等），分离 latin 和 ea typeface。

## 排版规范

- 标题: 28-32px, bold, heading_color
- 正文: 16-18px, text_color, 行高 dy=24
- 辅助文字: 14px
- 页码: 12px
- 要点: 用 ● 或数字序号
- 中文左对齐，每行最多 18-20 字
- 超出用 `<tspan>` 换行

# 语文低年级 · 童趣启蒙风

## 适用范围
- 面向一二年级语文启蒙教学。
- 适合识字、拼音、朗读、看图说话、课堂导入、闯关练习、课堂总结。
- 关键词：童趣、轻松、圆角、大留白、清晰分区、儿童友好、图文结合。

## 配色参考
以下配色应直接参考同目录 `metadata.xml` 中的 `visual.palette`，不要另起一套颜色体系。

- `primary_color`: `#4B8BFF`
- `secondary_color`: `#66CDAA`
- `accent_color`: `#FFC857`
- `card_bg_color`: `#FFFFFF`
- `secondary_bg_color`: `#EEF7FF`
- `text_color`: `#394150`
- `heading_color`: `#24507A`

说明：
- `background_prompt` 由上游统一生成逻辑控制，这里只提供配色参考，不要求在本文件里改背景生成逻辑。
- 页面中的装饰色以 `primary / secondary / accent` 为主，不要引入高饱和新颜色。
- 整体视觉要轻、亮、通透，不要做整页厚重蒙版。

## 通用视觉语言
- 画布固定为 `1280 × 720`，顶部有 `6px` 色条。
- 核心容器是大圆角卡片，卡片之间留足呼吸感。
- 装饰元素以浅色圆点、星形、胶囊标签为主，不使用科技感线框或复杂渐变。
- 标题、副标题保持短句，优先让低年级教师和学生一眼扫读完成。
- 如果模板中没有真实 `<image>` 槽位，就不要额外补图。
- 如果模板中存在 `<image>` 槽位，必须沿用其位置语义和长宽比，允许小幅微调，但不能改成完全不同的图形结构。

## 使用原则
- 模板是页面特征参考，不是逐像素复刻目标。
- 卡片数量、图片数量、文本条数优先按 `min-max` 区间理解，不要因为模板示例里有几张卡就强制规划成几张卡。
- 先完成教学内容策划，再选更匹配的版式；不要让版式反向绑死内容。
- 如果页面是 `quiz` / `exercise` 的答案揭晓页，必须保留原题页版式，只新增答案高亮层，不能破坏伪动画切换感。

## 页面模板说明

### `cover.svg`
- 命中关键词：封面、导入、开场、课程标题、课文题目。
- 页面特点：中央主卡承载标题，底部两张对称插图，两个胶囊标签做轻量补充。
- 图片参考：固定 `2` 张，左右对称，比例约 `11:8`。
- 使用建议：适合一眼建立课程主题，不要改成单张大图封面。

### `toc_1.svg`
- 命中关键词：目录、学习任务、学习内容、本课安排、学习步骤、单列目录。
- 页面特点：左图右目录，右侧是纵向目录卡列表。
- 图片参考：固定 `1` 张，位于左侧竖向图片区，比例 `3:4`。
- 内容建议：目录项推荐 `3-5` 项，保持短句和导航感。

### `toc_2.svg`
- 命中关键词：目录、学习任务、学习内容、本课安排、学习步骤、双列目录、目录概览。
- 页面特点：双列目录卡并列排布，不再使用左侧配图。
- 图片参考：固定 `0` 张。
- 内容建议：目录项推荐 `6-8` 项，适合条目较多的目录概览页。

### `section.svg`
- 命中关键词：章节过渡、新任务开始、第二部分、第三部分、学习任务。
- 页面特点：居中章节卡 + 顶部胶囊提示条，用于节奏切换。
- 图片参考：当前模板无真实图片槽位。
- 使用建议：保持轻过渡，不要为了填充版面强加插图。

### `content_mixed_grid.svg`
- 命中关键词：观察、说一说、看一看、知识点、互动、主图、任务。
- 页面特点：左侧主图，右上知识卡，右下表达/互动卡，左下可有 `1-2` 枚补充小卡。
- 图片参考：固定 `1` 张，位于左侧主图区。
- 适合内容：常规图文讲解页，“一张主图 + 两块教学信息”。

### `content_bento_2col_equal.svg`
- 命中关键词：两个角色、两类事物、左右对照、并列比较、两组信息、双栏。
- 页面特点：双列等宽，上图下文，两栏阅读权重一致。
- 图片参考：固定 `2` 张，每栏 `1` 张。
- 适合内容：两个人物、两个对象、两组并列特征。

### `content_bento_2col_asymmetric.svg`
- 命中关键词：主图配讲解、一主一辅、重点解析、侧边提示、图文解读、主次分明。
- 页面特点：左侧高幅主图，右上是 hero 讲解卡，右下可放 `1-2` 个补充卡。
- 图片参考：固定 `1` 张，左侧高幅。
- 适合内容：主信息很明确，需要“主图 + 深讲 + 提示”的页面。

### `content_bento_3col.svg`
- 命中关键词：三部分、三点归纳、三个角色、三个发现、三步、三栏。
- 页面特点：三列等宽，上图下文，节奏整齐。
- 图片参考：固定 `3` 张，每栏 `1` 张。
- 适合内容：三点归纳、三对象观察、三步讲解。

### `content_cards_top_hero_bottom.svg`
- 命中关键词：先归纳后看图、上方卡片、下方大图、先说要点、情境收束。
- 页面特点：上方 `2-4` 张短卡，下方一张大图承接情境。
- 图片参考：固定 `1` 张，位于下方 hero 区。
- 适合内容：先布置观察任务，再用大图承接。

### `content_hero_top_cards_bottom.svg`
- 命中关键词：情境导入、上图下卡、先看图再说、看图归纳、大图导入。
- 页面特点：上方大图导入，下方 `2-4` 张短卡补充知识点或任务。
- 图片参考：固定 `1` 张，位于上方 hero 区。
- 适合内容：先建立场景，再做分点整理。

### `content_hero_with_microcards_1.svg`
- 命中关键词：识字卡片、生字、拼音、组词、提示卡、并列小任务、观察提示、练习项。
- 页面特点：整页只有一张外层大卡，顶部说明条承载总述或任务，卡内按统一节奏排布 `2-6` 个微卡片。
- 图片参考：当前模板无真实 `<image>` 槽位，优先用卡内文字和小模块完成表达。
- 使用建议：适合一个主内容或总述，下方用 `2-6` 个短信息微卡展开；同一页内只能选一种微卡样式，不适合长段正文或复杂对比分析。

### `content_hero_with_microcards_2.svg`
- 命中关键词：诗歌、诗句、朗读、诵读、作者、短文、分栏短句、文本展示。
- 页面特点：一张居中大卡承载标题、作者和正文，`1-4` 行时用单列居中正文，`5-8` 行时切换为双列正文。
- 图片参考：当前模板无真实 `<image>` 槽位。
- 使用建议：适合古诗、古文、诗歌朗读和短文本展示；保持“单一主卡 + 卡内文本组织”，不要再补大图或额外外层小卡。

### `relation_1.svg`
- 命中关键词：关系图、结构关系、分类关系、因果关系、关联、构成。
- 页面特点：中心主卡位于画面中部，四个分支卡分布在四角，使用连线表达中心与各节点的关系。
- 图片参考：当前模板无真实 `<image>` 槽位。
- 使用建议：适合“一个中心概念 + 四个关联节点”的关系页，优先保持关系图结构，不要退化成普通项目符号列表。

### `exercise.svg`
- 命中关键词：练习、闯关、圈一圈、连一连、选一选、操作区、提示卡。
- 页面特点：顶部题目区，左侧操作区，右侧提示卡。
- 图片参考：`0-1` 张，可用于右侧提示卡顶部横图。
- 使用建议：题目页与答案揭晓页必须保持结构稳定，避免破坏伪动画切换。

### `quiz.svg`
- 命中关键词：选择题、填空题、判断题、小测、闯关、答题提示。
- 页面特点：三张题型卡横向排布，底部提示区横向展开。
- 图片参考：固定 `0` 张。
- 使用建议：重点放在题型清晰、选项稳定、答案层可叠加，不要补图。

### `summary.svg`
- 命中关键词：总结、回顾、本课收获、三点归纳、知识小结。
- 页面特点：三栏总结卡等宽排布，每栏顶部一张正方形图，下面是 `2-3` 条短总结。
- 图片参考：固定 `3` 张，比例 `1:1`。
- 使用建议：保持三栏结构，但每栏正文条数按内容灵活裁剪。

### `closing.svg`
- 命中关键词：收尾、再见、课后任务、今天的收获、结束语。
- 页面特点：中央收尾大卡，下半区三张并列回顾卡，底部一句鼓励性收束语。
- 图片参考：固定 `0` 张。
- 使用建议：以文字和回顾卡为主，不补图，保持温和收束。

## 生成约束
- 只在命中的单个 SVG 模板上做参考，不导入整目录所有模板。
- 图片位置和尺寸是参考，不是死规则；但长宽比和整体阅读方向要保持一致。
- 低年级页面要优先保证标题清晰、任务短句、教师讲解空间足够。
- 如果命中 `hero_with_microcards` 系列模板，页面主结构必须保持“一张外层大卡 + 卡内微模块”。
- 如果内容与模板示例结构不完全一致，优先调整卡片数量和文本密度，不要强行把内容塞进固定示例文案结构。

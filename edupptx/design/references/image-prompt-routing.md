# 图片风格 Prompt 路由规则

## 目标
- 图片生成 prompt 不再依赖旧的学科/年级模板目录名称。
- 当前模板家族只分为 `低年级`、`复用`、`高年级`。
- 学科、主题和内容气质通过页面文本与关键词继续细分。

## 路由顺序
1. 先按 `template_family` 选基础画风：
   `低年级` / `高年级` / `复用`
2. 再按主题关键词叠加子风格：
   识字、拼音、看图说话、数学图形、数学计算、古诗、比较分析等
3. 再附加 `role / page_type / aspect_ratio` 的构图约束
4. 最后统一补负向限制：
   无文字、无水印、无拼贴、背景不杂乱

## Family 基础风格

### 低年级
- 儿童教育绘本感
- 圆润造型
- 课堂友好、清晰、亲和
- 允许更明显的启蒙和操作感

### 复用
- 中性教育插画
- 结构清晰
- 跨学科可复用
- 不带明显低龄或高冷编辑感

### 高年级
- 编辑感教学插画
- 克制、安静、层级清晰
- 适合分析、对照、阅读与总结

## 常见主题子风格

### 低年级 / 语文启蒙
- 识字、拼音、组词、生字学习
- 看图说话、观察提示、小故事导入
- 风格应偏绘本化、对象清晰、背景简洁

### 低年级 / 数学启蒙
- 图形认识、数数、加减法、应用题情境
- 风格应偏教具感、几何感、可操作感

### 高年级 / 阅读分析
- 古诗文、比较阅读、人物分析、主题归纳
- 风格应更克制，允许纸张、书卷、留白等文学或分析气质

### 复用 / 中性通用
- 不带强学科倾向的目录插画、章节插画、提示型小图
- 优先结构清晰和不喧宾夺主

## 使用原则
- `query` 负责“画什么”
- 路由规则负责“怎么画”
- family 只决定基础视觉气质，不直接决定学科
- 学科与主题主要由 `topic / title / content_points / design_notes` 触发
- 如果 family 是 `复用`，优先使用中性基础风格

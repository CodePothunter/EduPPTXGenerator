# 图片风格 Prompt 路由规则

## 目标
- 图片生成 prompt 不再依赖旧的学科/年级模板目录名称。
- 图片基础画风只由主模板家族决定：`低年级` 或 `高年级`。
- `复用` 是结构模板池，不是独立视觉风格；当页面实际使用 `复用` 模板时，图片风格必须继承已命中的主模板家族。
- 只有完全没有命中 `低年级` / `高年级` 主家族时，才允许使用 `复用` 的中性兜底风格。
- 学科、主题和内容气质通过页面文本与关键词继续细分。

## 路由顺序
1. 先从 `template_family` 解析主画风：
   `低年级+复用` 按 `低年级`；`高年级+复用` 按 `高年级`；单独 `复用` 才按中性兜底。
2. 再按主题关键词叠加子风格：
   识字、拼音、看图说话、数学图形、数学计算、古诗、比较分析等。
3. 再附加 `role / page_type / aspect_ratio` 的构图约束。
4. 最后统一补负向限制：
   无文字、无水印、无拼贴、背景不杂乱。

## Family 基础风格

### 低年级
- 儿童教育绘本感。
- 圆润造型。
- 课堂友好、清晰、亲和。
- 允许更明显的启蒙和操作感。

### 高年级
- 编辑感教学插画。
- 克制、安静、层级清晰。
- 适合分析、对照、阅读与总结。

### 复用兜底
- 仅在没有主模板家族时使用。
- 中性教育插画，结构清晰，不带明显低龄或高冷编辑感。
- 不应覆盖或弱化已经命中的 `低年级` / `高年级` 风格。

## 常见主题子风格

### 低年级 / 语文启蒙
- 识字、拼音、组词、生字学习。
- 看图说话、观察提示、小故事导入。
- 风格应偏绘本化、对象清晰、背景简洁。

### 低年级 / 数学启蒙
- 图形认识、数数、加减法、应用题情境。
- 风格应偏教具感、几何感、可操作感。

### 高年级 / 阅读分析
- 古诗文、比较阅读、人物分析、主题归纳。
- 风格应更克制，允许纸张、书卷、留白等文学或分析气质。

### 复用模板的处理
- 如果页面模板来自 `复用`，但主家族已经命中 `低年级`，图片继续使用低年级绘本/启蒙风格。
- 如果页面模板来自 `复用`，但主家族已经命中 `高年级`，图片继续使用高年级克制/编辑感风格。
- 不因为实际使用了 `复用` 模板，就额外叠加一个新的“复用风格”。

## 使用原则
- `query` 负责“画什么”。
- 路由规则负责“怎么画”。
- 主 family 只决定基础视觉气质，不直接决定学科。
- 学科与主题主要由 `topic / title / content_points / design_notes` 触发。
- `复用` 只作为结构复用和无主家族时的中性兜底，不作为低年级/高年级之外的第三套风格。
